# Examples

This directory contains example files and test scripts for the Universal Name Formatter.

## Files

- `test-node.js` - Node.js example showing basic usage and testing
- `test-api.js` - API testing script for development

## Usage

To run the Node.js example:

```bash
npm run build
node examples/test-node.js
```

To run the API test:

```bash
npm run build
node examples/test-api.js
```

## Note

These files are for development and demonstration purposes. They are not part of the published npm package.
